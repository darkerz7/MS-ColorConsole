## CVARs:
Cvar | Parameter | Description
--- | --- | ---
`ms_colorconsole_chattimer` | <0/1> | Disabled/enabled timer in chat
`ms_colorconsole_hudtimer` | <0/1> | Disabled/enabled timer in hud(show_survival_respawn_status)
`ms_colorconsole_maxelement` | <1/10> | Maximum number of elements to show in the hud
`ms_colorconsole_maxtime` | <5/3600> | Maximum time for displaying the timer
